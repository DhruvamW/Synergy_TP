# Task 3 — Foundations of Data Analysis

## Objective

This task builds the theoretical base needed before moving into pandas, feature engineering, visualization, and machine learning. It covers what a dataset represents, how observations and variables differ, how categorical and numerical data differ, and why units and domain context change what a value means.

## Learning Outcome

By the end of this task, the following should be explainable without referring back to notes:

- Observations vs. variables
- Categorical vs. numerical data
- Raw data vs. processed data
- Measurement units and why they matter
- Basic summary statistics: mean, median, minimum, maximum, range
- When the mean can be misleading
- What to check before analyzing a dataset
- What to ask a team before analyzing data they hand off
